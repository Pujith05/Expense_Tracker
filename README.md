Expense Tracker

A simple and user-friendly React-based application to track daily expenses with features like add, edit, delete, and category management.

-> Features

* Add new expenses

* Edit existing expenses

* Delete expenses

* Shows date, description, category, and amount

* Organizes expenses in a table format

* React component-based structure

-> Tech Stack

* React.js

* JavaScript (ES6+)

* HTML & CSS

* Node.js (for development environment)
